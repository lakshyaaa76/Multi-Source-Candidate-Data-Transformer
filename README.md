# Candidate Transformer

A Python prototype for transforming candidate data from multiple source formats into a single canonical profile.

## Overview

This project is designed to:

- ingest structured and unstructured candidate data sources (CSV, ATS JSON, resume PDF, recruiter notes)
- normalize and merge conflicting fields into one canonical candidate record
- preserve provenance and confidence for merged values
- support a runtime projection config that selects, renames, and validates output fields

## Status

This is a working prototype intended to demonstrate pipeline design, source handling, merge logic, and configurable output projection. The README will be expanded later with installation and usage details.
